## 📝 지원 시 강조 포인트

### native-video-editor 프로젝트 연계

| 기업 유형 | 강조할 포인트 |
|-----------|---------------|
| **영상/미디어 기업** | FFmpeg C API 직접 사용, 썸네일 추출 p99 < 50ms, 영상 편집 도메인 이해 |
| **스트리밍 서비스** | 실시간 WebSocket, 성능 최적화, GPU 가속 (WebGL) |
| **일반 IT 스타트업** | 풀스택 역량, React/TypeScript, 프로덕션 배포 경험 |
| **대기업** | 체계적 문서화, 테스트 커버리지, 아키텍처 설계 |
