## 📝 이력서 기술 방법

### 프로젝트 요약 (2-3줄)

```
[C++ PVP Server] 프로덕션 레벨 1v1 실시간 대전 게임 서버
- C++17/Boost.Asio 기반 60 TPS 서버 권위 게임 루프 구현
- UDP 넷코드 (클라이언트 예측, 리컨실리에이션, 지연 보상)
- ELO 매치메이킹, Redis 분산 세션, 안티치트 시스템
```

### 기술 스택 기술

```
Language: C++17
Networking: Boost.Asio, Boost.Beast, UDP, gRPC
Database: PostgreSQL 15, Redis 7
Serialization: Protocol Buffers
Monitoring: Prometheus, Grafana
Build: CMake, vcpkg, Docker
```

### 성과 강조

```
• 60 TPS 게임 루프, 틱 편차 0.04ms (목표 1ms의 4%)
• WebSocket p99 지연 18.3ms (목표 50ms)
• ELO 매치메이킹 200명 < 2ms 처리
• UDP 델타 압축으로 대역폭 80% 절감
• Consistent Hash 로드 밸런서, 최소 재할당
```

## ⚠️ 게임 업계 면접 특수 준비

### 기술 면접 예상 질문

1. **게임 루프**: 고정 vs 가변 시간 단계 차이
2. **충돌 감지**: AABB vs OBB vs 원-원
3. **넷코드**: 입력 지연 vs 롤백 vs 예측
4. **동시성**: Lock-free 자료구조
5. **메모리**: 오브젝트 풀, 캐시 친화적 설계

### 코딩 테스트 예상 유형

1. 알고리즘: 그래프, DP (일반)
2. 공간 분할: 쿼드트리, 옥트리
3. 수학: 벡터, 행렬, 삼각함수
4. 동시성: 생산자-소비자, 락 프리

### 과제 전형 예상

1. 간단한 멀티플레이어 게임 서버 구현
2. 충돌 감지 시스템 최적화
3. 매치메이킹 알고리즘 구현
