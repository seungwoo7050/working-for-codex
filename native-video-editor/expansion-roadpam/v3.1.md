# native-video-editor v3.1 WebGL 기반 비디오 편집

## 프로젝트 개요

### 현재 상태 (v3.0.4)
- **위치**: `/native-video-editor/`
- **기술 스택**: C++, OpenGL, FFmpeg
- **완성된 기능**:
  - 기본 비디오 편집
  - GPU 가속 렌더링
  - 실시간 프리뷰

### 목표 상태 (v3.1 완료)
- **목표**: WebGL 기반 웹 브라우저 비디오 편집
- **핵심 변경**: 브라우저 포팅, WebGL API 활용

## 기술 스택 추가

- **WebGL**: WebGL 2.0, GLSL ES 3.0
- **비디오 처리**: HTML5 Video, Canvas 2D API
- **모던 JS**: ES6+, Web Workers, WebAssembly

## 성능 요구사항

- 60 FPS 실시간 프리뷰
- 4K 비디오 지원
- 브라우저 메모리 < 500MB

## 브라우저 지원

- Chrome 51+
- Firefox 45+
- Safari 10+
- Edge 79+
