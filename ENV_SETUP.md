# ENV_SETUP – Toy Project Suite

## 공통 사전 준비
- Git 2.30+
- Java 17 (Spring Boot용)
- Node.js 18+ 및 npm 9+
- g++ (C++17 지원)

## 운영체제별 힌트
- Ubuntu: `sudo apt-get install openjdk-17-jdk nodejs npm g++`
- macOS(Homebrew): `brew install openjdk@17 node g++`

## 프로젝트별 실행 포트
- 백엔드: 8080
- 프론트엔드: 3000
- echo-server: 9000 (기본)
- multi-chat-server: 9100 (예시)

## 빠른 점검 명령어
```bash
java -version
node -v
npm -v
g++ --version
git --version
```

## 태그 재현 예시
```bash
git checkout BE-v0.1
```
