## 🎯 핵심 기술 역량 매핑

### 역량 1: 시스템 프로그래밍 (C++/Native)

| 증명 포인트 | 상세 내용 |
|------------|----------|
| C++ N-API 바인딩 | Node.js ↔ C++ 데이터 전송 최적화 |
| FFmpeg C API 직접 연동 | libavformat, libavcodec, libswscale |
| RAII 패턴 | AVFormatContextPtr, AVCodecContextPtr |
| 메모리 풀링 | 32-frame AVFrame 재사용 풀 |

**적합 포지션**: 미디어 엔진 개발, 게임 클라이언트, 시스템 SW

### 역량 2: GPU 프로그래밍 (WebGL/Shader)

| 증명 포인트 | 상세 내용 |
|------------|----------|
| WebGL2 컨텍스트 관리 | Context loss recovery 처리 |
| GLSL 셰이더 작성 | 밝기, 대비, 채도 이펙트 |
| Texture 관리 | Atlas 패킹, LRU 캐시 |
| Framebuffer 파이프라인 | 멀티패스 렌더링 |

**적합 포지션**: 그래픽스 엔진, 비디오 렌더러, 게임 개발

### 역량 3: 실시간 시스템 (WebSocket/WebAudio)

| 증명 포인트 | 상세 내용 |
|------------|----------|
| WebSocket 양방향 통신 | 실시간 진행률 업데이트 |
| WebAudio 노드 그래프 | GainNode, AnalyserNode, BiquadFilter |
| AudioWorklet | 저지연 오디오 처리 |
| 프레임 동기화 | 비디오-오디오 정확 동기화 |

**적합 포지션**: 스트리밍 서비스, 실시간 협업, 미디어 플랫폼

### 역량 4: 풀스택 개발 (React/Node.js/Infra)

| 증명 포인트 | 상세 내용 |
|------------|----------|
| React 18 + TypeScript | 모던 프론트엔드 |
| Node.js + Express | RESTful API 설계 |
| PostgreSQL + Redis | 영속화 + 캐싱 |
| Docker Compose | 6개 서비스 오케스트레이션 |
| Prometheus + Grafana | 관측성 구현 |

**적합 포지션**: 풀스택 개발, 백엔드 개발, DevOps

```

Native: C++17, N-API, FFmpeg C API
Infra: Docker, Prometheus, Grafana, GitHub Actions
```

```
• 썸네일 추출 p99 47ms 달성 (목표 50ms)

• 6개 서비스 Docker Compose 오케스트레이션
• Prometheus/Grafana 통합 모니터링 구축
```
