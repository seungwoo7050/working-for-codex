현재 레포지토리는 두 개의 프로젝트를 포함하고있습니다.

각각의 프로젝트는 독립적으로 빌드 및 실행될 수 있는 완전히 분리된 프로젝트입니다.

1. `spring-patterns/` - Spring 기반의 백엔드 서비스 프로젝트
2. `native-video-editor/` - 네이티브 기반의 비디오 편집기 프로젝트
3. `modern-irc` - C++17 기반의 IRC 서버 프로젝트
4. `ray-tracer` - C++17 기반의 레이 트레이싱 렌더러 프로젝트

두 프로젝트의 최종 버전정보는 CLONE-GUIDE.md 파일에 명시되어 있습니다.

각 프로젝트의 유닛/빌드/e2e 테스트를 시행 후 정상 동작하는지 확인하시기 바랍니다.

또한 모든 테스트를 통과한 후, design/ 디렉터리의 설계 문서들과 실제 소스코드 상태가 일치하는지 검증하시기 바랍니다.

마지막으로 CI/CD 파이프라인이 정상적으로 동작하는지 확인하고 업데이트 사항이 있다면 반영하시기 바랍니다.
